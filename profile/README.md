## 💻 웹 서비스 소개

Zoom 로그 파일(.csv)을 업로드하면 참석자의 총 참여 시간을 자동으로 분석하고 합산해 줍니다. <br/>
이를 통해 비교과 활동 담당자는 행사 이수 여부를 간편하게 확인하고, 이를 파일(.xlsx)로 다운로드할 수 있습니다.
